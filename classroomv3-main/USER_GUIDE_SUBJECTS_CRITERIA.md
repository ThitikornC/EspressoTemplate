# 🎯 คู่มือใช้งานระบบจัดการวิชาและเกณฑ์การประเมิน

## สิ่งที่เพิ่มใหม่

### ✅ สามารถจัดการวิชาได้เอง
- เพิ่มวิชาใหม่
- แก้ไขชื่อวิชา
- ลบวิชา
- วิชาเริ่มต้น: วิทยาศาสตร์, คณิตศาสตร์, ภาษาไทย, สังคม, ภาษาอังกฤษ, พละศึกษา, ศิลปะ

### ✅ สามารถจัดการเกณฑ์การประเมินได้เอง (แยกตามวิชา)
- แก้ไขหัวข้อเกณฑ์
- เพิ่มเกณฑ์ใหม่
- ลบเกณฑ์
- รีเซ็ตกลับค่าเริ่มต้น
- เกณฑ์เริ่มต้น: 15 ข้อ (ตรงต่อเวลา, มีความรับผิดชอบ, ฯลฯ)

## วิธีใช้งาน

### 📚 จัดการวิชา

1. **เปิด Modal จัดการวิชา**
   - คลิกปุ่ม "📚 จัดการวิชา" ที่มุมขวาบน
   - จะเห็นรายการวิชาทั้งหมด

2. **เพิ่มวิชาใหม่**
   - คลิกปุ่ม "➕ เพิ่มวิชาใหม่"
   - กรอกชื่อวิชาใหม่ในกล่องที่โผล่มา
   - คลิก OK

3. **แก้ไขชื่อวิชา**
   - คลิกปุ่ม ✏️ ที่วิชาที่ต้องการแก้ไข
   - กรอกชื่อใหม่
   - ข้อมูลการประเมินจะถูกย้ายไปยังชื่อใหม่อัตโนมัติ

4. **ลบวิชา**
   - คลิกปุ่ม 🗑️ ที่วิชาที่ต้องการลบ
   - ถ้ามีข้อมูลการประเมิน จะมีการเตือน
   - ยืนยันการลบ (ข้อมูลจะหายไปด้วย)

### ✏️ แก้ไขเกณฑ์การประเมิน

1. **เข้าสู่โหมดแก้ไข**
   - คลิกปุ่ม "✏️ แก้ไขหัวข้อ"
   - จะเปลี่ยนเป็น "✅ เสร็จสิ้น"
   - หัวข้อในตารางจะคลิกได้

2. **แก้ไขหัวข้อ**
   - คลิกที่หัวข้อใดก็ได้
   - จะเห็น textarea ให้แก้ไข
   - แก้ไขแล้วคลิก "✅ บันทึก" หรือ "❌ ยกเลิก"

3. **เพิ่มหัวข้อใหม่**
   - ในโหมดแก้ไข คลิกปุ่ม "➕ เพิ่มหัวข้อ"
   - กรอกเกณฑ์การประเมินใหม่
   - คลิก OK

4. **ลบหัวข้อ**
   - ในโหมดแก้ไข จะเห็นปุ่ม 🗑️ ที่แต่ละหัวข้อ
   - คลิกเพื่อลบ (ต้องมีอย่างน้อย 3 หัวข้อ)

5. **รีเซ็ต**
   - คลิกปุ่ม "🔄 รีเซ็ต"
   - เกณฑ์จะกลับไปเป็นค่าเริ่มต้น 15 ข้อ

## ข้อควรระวัง ⚠️

1. **ต้องมีอย่างน้อย 1 วิชา** - ไม่สามารถลบวิชาสุดท้ายได้
2. **ต้องมีอย่างน้อย 3 เกณฑ์** - เพื่อให้การประเมินมีความหมาย
3. **ลบวิชา = ลบข้อมูล** - เมื่อลบวิชา ข้อมูลการประเมินทั้งหมดของวิชานั้นจะหายไป
4. **เกณฑ์แยกตามวิชา** - แต่ละวิชาสามารถมีเกณฑ์การประเมินต่างกันได้

## ตัวอย่างการใช้งาน

### กรณีที่ 1: เพิ่มวิชา "ดนตรี"
1. คลิก "📚 จัดการวิชา"
2. คลิก "➕ เพิ่มวิชาใหม่"
3. พิมพ์ "ดนตรี"
4. คลิก OK
5. ปิด modal → เลือกวิชา "ดนตรี" จาก dropdown

### กรณีที่ 2: แก้ไขเกณฑ์วิชาคณิตศาสตร์
1. เลือกวิชา "คณิตศาสตร์" จาก dropdown
2. คลิก "✏️ แก้ไขหัวข้อ"
3. คลิกที่หัวข้อ "ตรงต่อเวลา"
4. แก้เป็น "ส่งงานตรงเวลา"
5. คลิก "✅ บันทึก"
6. คลิก "✅ เสร็จสิ้น" เพื่อออกจากโหมดแก้ไข

### กรณีที่ 3: เพิ่มเกณฑ์ "มีความคิดสร้างสรรค์"
1. คลิก "✏️ แก้ไขหัวข้อ"
2. คลิก "➕ เพิ่มหัวข้อ"
3. พิมพ์ "มีความคิดสร้างสรรค์"
4. คลิก OK
5. เกณฑ์ใหม่จะปรากฏในตาราง

## การจัดเก็บข้อมูล

### LocalStorage Keys
- `subjects` - รายการวิชาทั้งหมด
- `criteria-วิทยาศาสตร์` - เกณฑ์การประเมินวิชาวิทยาศาสตร์
- `criteria-คณิตศาสตร์` - เกณฑ์การประเมินวิชาคณิตศาสตร์
- `evaluation-วิทยาศาสตร์` - คะแนนการประเมินวิชาวิทยาศาสตร์
- `evaluation-คณิตศาสตร์` - คะแนนการประเมินวิชาคณิตศาสตร์

### ข้อมูลที่บันทึก
```json
{
  "subjects": ["วิทยาศาสตร์", "คณิตศาสตร์", "ภาษาไทย"],
  "criteria-วิทยาศาสตร์": [
    { "id": "1", "description": "ตรงต่อเวลา" },
    { "id": "2", "description": "มีความรับผิดชอบ" }
  ],
  "evaluation-วิทยาศาสตร์": {
    "student-1": {
      "1": 3,
      "2": 2
    }
  }
}
```

## คำถามที่พบบ่อย (FAQ)

**Q: ถ้าลบวิชา ข้อมูลจะหายไหม?**
A: ใช่ ข้อมูลการประเมินทั้งหมดของวิชานั้นจะถูกลบด้วย (มีการเตือนก่อนลบ)

**Q: แก้ไขชื่อวิชาแล้ว ข้อมูลจะหายไหม?**
A: ไม่หาย ข้อมูลจะถูกย้ายไปยังชื่อใหม่อัตโนมัติ

**Q: แต่ละวิชาใช้เกณฑ์เดียวกันไหม?**
A: ไม่ แต่ละวิชาสามารถมีเกณฑ์การประเมินต่างกันได้

**Q: สามารถรีเซ็ตเฉพาะวิชาใดวิชาหนึ่งได้ไหม?**
A: ได้ คลิก "🔄 รีเซ็ต" จะรีเซ็ตเฉพาะวิชาที่เลือกอยู่

**Q: เกณฑ์การให้คะแนน (1-3) เปลี่ยนได้ไหม?**
A: ยังไม่ได้ในเวอร์ชันนี้ แต่สามารถพัฒนาต่อได้

## สรุป

ระบบนี้ให้ความยืดหยุ่นสูงในการปรับแต่งตามความต้องการของแต่ละโรงเรียน/ครู โดย:
- ✅ เพิ่ม/ลบ/แก้ไขวิชาได้เอง
- ✅ ปรับแต่งเกณฑ์การประเมินแยกตามวิชา
- ✅ ข้อมูลถูกบันทึกอัตโนมัติ
- ✅ มีการป้องกันข้อผิดพลาด
- ✅ UI ใช้งานง่าย สวยงาม

---

**หมายเหตุ**: ระบบนี้ใช้ localStorage ในการจัดเก็บข้อมูล ดังนั้นข้อมูลจะถูกเก็บไว้ในเบราว์เซอร์ของเครื่องที่ใช้งานเท่านั้น
